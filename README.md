# 3D Tetris Clone

# Sobre o projeto

Jogo desenvolvido na Unity 3D inspirado no clássico [Tetris](https://en.wikipedia.org/wiki/Tetris)

## Descrição
- O jogador deve empilhar as peças de forma que elas completem uma linha horizontal
- Contém 5 peças de cores diferentes
- Na tela existe um indicador que mostra a próxima peça e outro que mostra a pontuação atual
- Assets em 3D

## Objetivo
- O jogador deve empilhar as peças de forma que elas completem uma linha horizontal
- O jogador ganha pontos ao completar cada linha horizontal
- Se a peça encaixada enconstar no topo, o jogador perde

# Tecnologias utilizadas
- Unity 3D
- C#
